# 独龙狂舞 (Dragon Bench Dance)

手机竖屏操作的 2D 动作生存小游戏，基于 React + Vite + TypeScript + Canvas 2D。

## 本地开发

```bash
npm install
npm run dev
```

打开浏览器后使用手机模式或直接在手机上访问开发地址进行体验。

## 构建

```bash
npm run build
npm run preview
```

## 自动发布（GitHub + Vercel）

本仓库支持 Vercel 静态部署，构建命令与产物目录如下：

- Build Command: `npm run build`
- Output Directory: `dist`

连接 GitHub 仓库到 Vercel 后，推送到默认分支会自动触发发布。
